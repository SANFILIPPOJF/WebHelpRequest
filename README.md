# WebHelpRequest
Bienvenue sur le portail de gestion des tickets de demande d'aide realisez par Jean-Sébastien et Jean-François

Cette page permet d'envoyé une demande d'aide à Philippe
pour cela, selectionnez l'apprenant.e puis décrivez votre besoin dans la case prevue a cette effet.
puis cliquez sur le bouton "je veux de l'aide"

Vous pouvez aussi consulter les statistiques de chaque apprenant.e.
Pour cela selectionnez l'apprenant et cliquez sur "statistique apprenant"

Vous pourrez consulter la liste des tickets en attente de resolution.
Un simple click sur "poubelle" correspondant a un ticket signifira que celui ci est devenu caduque.
